# Intro #

Makes a "tag cloud" using [D3.js](http://d3js.org).

Try this:

```bash
git clone https://github.com/mattjj/tagcloud.git
cd tagcloud
python -m SimpleHTTPServer
# point your browser to localhost:8000/example/
```
